Overview

This is a lecture-based course that will introduce to you to the key concepts you should know in order to authorize an application and authenticate a user to access data stored on an OAuth 2.0 projected server.

Objectives

  1. Describe the benefits of OAuth 2.0
  2. Register your app with an OAuth 2.0 server
  3. Choose an OAuth 2.0 flow for your mobile app

Lecture

ENT171 - Introduction to OAuth 2.0.
